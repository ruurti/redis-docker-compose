# Redis Server - Redis Insight

Triển khai Redis Server và trình quản trị Redis bằng Redis Insight

---

## Clone repo về máy local

```shell
git clone https://github.com/ruurti/redis-sample
```

---

## Redis Single


### Cài đặt 

```shell
cd redis_alone && docker compose up --build -d
```
