# Single Node Redis

## Cài đặt 

```shell
docker compose up --build -d
```

## Thông tin kết nối

* Redis Official:
    * Host: 127.0.0.1
    * Port: 6379
    * Username / Password: Rỗng

* Redis Bitnami: (Mặc định vô hiệu hoá trong `docker-compose.yaml`)
    * Host: 127.0.0.1
    * Port: 16379
    * Username / Password: Rỗng

## Redis Insight 

* Redis Official: http://localhost:5540/redis-official-local/browser
